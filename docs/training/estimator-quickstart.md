# Estimator Quick Start Guide

## Getting Started

1. **Login** at `app.krewpact.com` with your MDM email
2. Navigate to **Estimates** from the sidebar

## Creating an Estimate

### Step 1: New Estimate

1. Click **New Estimate**
2. Fill in:
   - **Project name** or reference
   - **Division** (Contracting, Homes, etc.)
   - **Client/Account** (link to CRM account)
   - **Due date** for the bid

### Step 2: Line Items

1. Click **Add Line Item**
2. For each item, enter:
   - **Description** — What the work is
   - **Cost code** — Select from the cost code library
   - **Quantity** and **unit** (SF, LF, EA, LS, etc.)
   - **Unit price** — Material + labor rate
   - **Markup %** — Applied automatically per division settings
3. Subtotals and grand total calculate automatically

### Step 3: Assemblies

- Use **Assemblies** to insert pre-built groups of line items
- Navigate to **Assemblies** in the sidebar to create reusable assemblies
- Each assembly contains standard items with default quantities and pricing

### Step 4: Allowances & Alternates

- **Allowances** — Add provisional sums (e.g., "Electrical allowance: $50,000")
- **Alternates** — Add pricing options the client can choose (e.g., "Upgrade to hardwood: +$12,000")

### Step 5: Version Control

- Click **Save Version** to create a snapshot before making changes
- View all versions in the **Versions** tab
- Compare versions to see what changed

## Estimate Templates

- **Estimate Templates** — Save frequently used estimate structures
- Apply a template when creating a new estimate to pre-populate line items

## Cost Reference

- **Cost Catalog** — Browse historical pricing data
- **Cost Codes** — View the full cost code hierarchy with ERPNext mappings

## Workflow

1. Create estimate → Add line items → Save version
2. Submit for **review** to Operations Manager
3. Once **approved**, PM creates a Proposal for the client
4. Signed proposal → Project creation

## Tips

- Use assemblies for repetitive scopes (e.g., "Standard bathroom rough-in")
- Always save a version before major pricing changes
- Check the cost catalog for recent pricing benchmarks
- Link estimates to CRM opportunities for pipeline tracking

## Need Help?

Contact your Operations Manager or Project Manager.
