Scheduled Cron Jobs

JobScheduleDescription
apollo-pumpDailyPulls new prospect data from Apollo.io into lead pipeline
enrichmentPeriodicRuns waterfall enrichment on leads missing data (Apollo → Clearbit → LinkedIn → Google)
sequence-processorPeriodicExecutes pending outreach sequence steps (emails, follow-ups)

Feature Modules

Phase 1 — Core CORE

Auth & Identity

Clerk SSO (M365 groups: MDM Group, Contracting, Wood, Telecom) + email/password fallback

RBAC

13 canonical roles, division-level access control

Division Management

5 MDM divisions with isolated data access

CRM: Leads

Lead CRUD, 6-stage pipeline, stage history

CRM: Opportunities

Opportunity CRUD, 7-stage pipeline, conversion from leads

CRM: Accounts & Contacts

Company and contact management with relationships

CRM: Activities

Activity logging (calls, emails, meetings, notes)

Lead Scoring

AI-powered fit + intent + engagement scoring engine

Outreach Sequences

Multi-step automated outreach with configurable delays

Enrichment Pipeline

Waterfall: Apollo → Clearbit → LinkedIn → Google

Estimating

Estimate builder, line items, sections, version history

Projects

Project CRUD, team members, milestones, tasks

Daily Logs

Site diary entries with weather, crew, work performed

Expenses

Expense CRUD with approval workflow and claims

Calendar & Scheduling

Calendar events API + schedule page

Notifications

Engine with in-app, email, push, and SMS channels

Email (Resend)

Compose, send, templates via Resend API

Reports

Lead sources, pipeline views, dashboard metrics

ERPNext Sync

12 entity mappings, BullMQ async sync via Cloudflare Tunnel

Team Management

Team page + API for member management

Dashboard

Metrics dashboard with KPIs and quick actions

Admin: Lead Gen

Admin lead generation page for bulk operations
